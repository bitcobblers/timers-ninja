# Getting Started

Timres.ninja is build with these technologies:

- [Node.js](https://nodejs.com)
- [Qwik](https://qwik.dev)
- [TailwindCSS](https://tailwindcss.com)
- [Chevrotrain](http:/someurl.dev)

To get started:

```shell 
git clone https://github.com/bitcobblers/timers-ninja
npm install
```

To run the application locally:

```shell
npm run dev
```

To run the storybook locally:

```shell
npm run storybook
```

To run the Unit Tests:

```shell
npm run test.unit
